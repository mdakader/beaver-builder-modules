    A simple plugin for Beaver Page Builder Modules.

== Description ==

    Easy Modules plugin will add features For Beaver Builder Modules.


= Available Modules =

<ul>
    <li>Image Gallery</li>
    <li>Dual Heading</li>
    <li>Flip Box</li>
    <li>Services</li>
    <li>Team</li>
    <li>Testimonial</li>
    <li>Timeline</li>
    <li>Call to action</li>
</ul>



== Installation ==

1. Install Easy Modules either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

2. After activating Easy Modules, navigate to the Beaver Modules.
